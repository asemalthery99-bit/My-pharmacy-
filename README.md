# IBPharmacy Pro - منصة صيدلية إلكترونية متكاملة

## 🏥 نظرة عامة

**IBPharmacy Pro** هي منصة صيدلية إلكترونية حديثة وشاملة تجمع بين تقنيات الويب المتقدمة والذكاء الاصطناعي لتوفير تجربة استخدام استثنائية للمرضى والصيادلة.

## ✨ المميزات الرئيسية

### 👥 للمرضى
- **كتالوج أدوية متقدم** مع بحث وفلترة حسب الفئة والسعر والتوفر
- **صفحات تفاصيل شاملة** للأدوية مع معلومات عن الاستخدام والجرعة والتحذيرات
- **سلة تسوق ذكية** مع إدارة الكميات وحساب الإجمالي
- **نظام طلبات متقدم** مع تتبع حالة الطلب في الوقت الفعلي
- **رفع الوصفات الطبية** مع معالجة الصور بالذكاء الاصطناعي
- **روبوت محادثة AI** للإجابة على استفسارات حول الأدوية والتفاعلات الدوائية

### 💼 للصيادلة
- **لوحة تحكم شاملة** لإدارة الطلبات والوصفات الطبية
- **إدارة المخزون** والأدوية المتوفرة
- **معالجة الوصفات الطبية** والموافقة عليها
- **تقارير وإحصائيات** مفصلة عن الطلبات والمبيعات

## 🛠️ التقنيات المستخدمة

### Frontend
- **React 19** - مكتبة واجهات المستخدم
- **TypeScript** - لغة برمجة آمنة النوع
- **Tailwind CSS 4** - تصميم واستجابة
- **Vite** - أداة بناء سريعة وحديثة
- **shadcn/ui** - مكونات واجهة مستخدم عالية الجودة

### Backend
- **Express.js** - خادم ويب قوي
- **tRPC** - نظام استدعاء الإجراءات الآمن النوع
- **Node.js** - بيئة تشغيل JavaScript

### Database
- **MySQL/TiDB** - قاعدة بيانات علائقية
- **Drizzle ORM** - أداة تعامل مع قاعدة البيانات

### AI & Services
- **OpenAI API** - معالجة الصور واستخراج النصوص
- **Manus OAuth** - نظام مصادقة آمن
- **Google Maps API** - خدمات الخرائط

## 🚀 البدء السريع

### المتطلبات
- Node.js 18+
- pnpm أو npm
- قاعدة بيانات MySQL

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/asemalthery99-bit/My-pharmacy-.git
cd My-pharmacy-

# تثبيت الاعتماديات
pnpm install

# إعداد متغيرات البيئة
cp .env.example .env.local

# تشغيل خادم التطوير
pnpm dev
```

### النشر على Vercel

1. ادخل إلى [Vercel](https://vercel.com)
2. اختر "Import Project"
3. اختر المستودع `asemalthery99-bit/My-pharmacy-`
4. أضف متغيرات البيئة المطلوبة
5. انقر "Deploy"

## 📖 التوثيق

### صفحات رئيسية
- **الصفحة الرئيسية** (`/`) - عرض الأدوية المميزة
- **الكتالوج** (`/catalog`) - قائمة جميع الأدوية
- **تفاصيل الدواء** (`/medicine/:id`) - معلومات شاملة
- **السلة** (`/cart`) - إدارة الطلبات
- **الطلبات** (`/orders`) - تاريخ الطلبات
- **الوصفات** (`/prescriptions`) - رفع ومعالجة الوصفات
- **الدردشة** (`/chat`) - روبوت المحادثة
- **لوحة التحكم** (`/dashboard`) - للصيادلة
